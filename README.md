# androidx86-ROOT
All androidx86 ROOT<br>
like  Phoenix OS , REMIX OS, Prime OS etc.<br>
wrok for ROOT<br>


32bit<br>
cd /system/xbin<br>
mv su su.orig<br>
cp /sdcard/su/x86/su.pie su<br>
chmod 4751 su<br>
su --install<br>
su --daemon<br>

64bit<br>
cd /system/xbin<br>
mv su su.orig<br>
cp /sdcard/su/x64/su .<br>
chmod 4751 su<br>
su --install<br>
su --daemon<br>
